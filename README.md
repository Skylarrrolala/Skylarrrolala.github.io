# Skylar's Portfolio — skylar-thedev.me

Personal portfolio for Dararithy Heng (Skylar) — CS student, AI builder, aspiring Product Manager based in Phnom Penh, Cambodia.

**Live site:** [Skylarrrolala.github.io](https://Skylarrrolala.github.io)

---

## Stack

- **HTML / CSS / Vanilla JS** — no build step, deploys directly to GitHub Pages
- **Tailwind CSS** via Play CDN
- **Three.js** (ES module via importmap) — interactive 3D lanyard in hero section
- **Web3Forms** — contact form backend

## Structure

```
├── index.html              # Main page
├── assets/
│   ├── css/
│   │   └── style.css       # All custom styles
│   ├── js/
│   │   ├── main.js         # Layout, scroll, animations, contact form
│   │   └── lanyard.js      # Three.js interactive lanyard (physics + 3D)
│   └── images/
│       ├── logos/          # School and org logos
│       └── skills/         # Skill icon images
└── CNAME                   # Custom domain config
```
